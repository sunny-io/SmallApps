Eviltester Calculator

Page tested: page url

Selenium Webdriver file: calculator.js

Assertion library: Chai

Testcases

Table of human readable testcases: Google Sheet

Testcases as JS file: testdata_calculator.js

Testcase Structure

Testcases is a list of testcase objects.

Testcase Object Description

  • name: testcase identifier, string
  • field1: input data for first input field, number or string
  • field2: input data for second input field, number oder string
  • operator: text to be selected in the function selection list, string
  • expected: Expected result, number or string
  • description: short description of test case, string
How to Run Test

Run calculator.js to execute all tests.

Locator Strategy

IDs are used as locators.

Function Description
runAllTests(testcases)

The function loops through testcases and calls test(testcase) for each.

Each test is run in its own browser instance.

test(testcase)

The function generates the driver and browser instance for the test case and calls the function fillFieldsAndCheck(driver, testcase) executing data input and assertion.

fillFieldsAndCheck(driver, testcase)

This function executes the test case and asserts the result:

  1. Calls fillField(driver, fieldId, keys ) for each field
  2. Calls doSelect(driver, operator) to set the correct operator
  3. Clicks on the Calculate button
  4. Asserts the result
fillField(driver, fieldId, keys)

This function fills the field and asserts the field value after input.

doSelect(driver, operator)

This function creates a Select object to interact with the selection list.

  1. Creates Select object named 'select'
  2. Uses select.selectByVisibleText(operator) to set option
  3. Gets the text of the selected option using select.getFirstSelectedOption() followed by .getText() on the result
  4. Asserts selected option matches operator
